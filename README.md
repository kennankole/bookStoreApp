# 📖 [Books Store] <a name="about-project"></a>

**[Books Store]** is an online bookstore build with React and Redux. Books can be added and removed from the app.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 install node
```

### Setup

```sh
  cd my-folder
  git clone https://github.com/kennankole/bookStoreApp.git
```

### Install

Install this project with:

```sh
  cd bookStoreApp
  npm install
```

### Usage

```sh
  npm start
```

### Run tests

To run tests, run the following command:

```sh
 npm test
```
